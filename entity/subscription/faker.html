Railken\Amethyst\Fakers\SubscriptionFaker

The faker can be used for testing or seeding.

Create a new entity using the faker

use Railken\Amethyst\Fakers\SubscriptionFaker;

$result = $manager->create(SubscriptionFaker::make()->parameters());

Extend the class

Create the new faker in app/Fakers/SubscriptionFaker

namespace App\Fakers;

use Railken\Amethyst\Fakers\SubscriptionFaker as Faker;

class SubscriptionFaker extends Faker {
    // ...
}

Update the file configs/amethyst.subscription with the new class

return [
    'data' => [
        'subscription' => [
            'faker' => App\Fakers\SubscriptionFaker::class,
        ],
    ]
];
